Z_PD_EXTERN void z_pd_clear ( )

Clears the Pure Data environment.

The method should be called only one time when the Pure Data environment will never be called again.

Z_PD_EXTERN void z_pd_init ( )

Initializes the Pure Data environment.

The method should be called only one time at the initialization, before everything else.
