Detailed Description

A symbol is used as for fast comparaison of string.

A symbol is bound to a sting. The symbol on is global to all the xpd environment. At the creation, the method looks for the sting in a global table and return a reference to the symbol if name is there, otherwise the method inserts the sting in the table and associates with this new symbol.
