Detailed Description

A tie is used as a connection to send and receive messages.

A tie is bound to a sting so it can be used to communicate with other objects that use this string. The tie on is global to all the xpd environment. At the creation, the method looks for the sting in a global table and return a reference to the tie if name is there, otherwise the method inserts the sting in the table and associates with this new tie. If you want to communicate with a specific patch, you should use the id of a patch.
