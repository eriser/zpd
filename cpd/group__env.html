void cpd_init ( )

Initializes the cpd environment.

The method should be called only one time at the initialization, before everything else.

void cpd_clear ( )

Clears the cpd environment.

The method should be called only one time when the Pure Data environment will never be called again.

void cpd_searchpath_clear ( )

Clears the search path of Pure Data.

Clears the search path for all the environment. The path will be used when you try to open a patch, to load an abstraction or an external.
