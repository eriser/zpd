cpd_instance* cpd_instance_new ( size_t  size)

Creates a new instance.

If you want to implement your own instance, the first memeber of the structure must be a cpd_instance, after this you can add everything you want. You should allocate your instance with this function using sizeof(myinstance).

struct myinstance
{
cpd_instance instance;
int* values;
};
...
myinstance* inst = cpd_instance_new(sizeof(myinstance));
if(inst)
inst->value = (float *)malloc(512 * sizeof(float));
...
Parameters
sizeThe size of memory to allocate in bytes.
Returns
A pointer to the initialized cpd_instance or NULL if the allocation failed.
void cpd_instance_free ( cpd_instance *  instance)

Deletes an instance.

You must first delete the members of your instance if need and therefater call this method to free the instance.

struct myinstance
{
cpd_instance instance;
int* values;
};
...
myinstance* inst;
..
free(inst->value);
cpd_instance_free(inst);
...
Parameters
instanceThe pointer to the instance.
