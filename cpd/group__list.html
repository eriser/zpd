enum cpd_listtype

The type of messages in a list.

Enumerator
CPD_NULL 

The element is undefined or null.

CPD_FLOAT 

The element is a floating point number.

CPD_SYMBOL 

The element is a symbol.

CPD_POINTER 

The element is a graphical pointer.
